# Adaption

:

**Jeder Charackter kann maximal 1 Adaptionpunkt besitzen, wie hoch sein Level ist. Wann immer ein Anrgiffswurf misslingt bekommst du 1 Adaptionpunkt. (Außer von einer Adaptionsfähigkeit)**

:

## Nahkampf Adaption

::

### Vorbereitung

> **1 Adaption / Nahkampf**

:

*Wenn ein anderes Ziel in Reichweite ist:* Greife das Ziel an. Triffst du bewirkst du kein Schaden, aber dieses Ziel darf diese Runde nicht Angreifen.

:

### Instant Tausch

> **3 Adaption / Dolch**

:

*Wenn du angegriffen wirst und ein anderes Ziel in Reichweite ist:* Greife das Ziel an. Triffst du tauschst du mit dem Ziel die position und das Ziel wird vom Angriff getroffen nicht du.

:

## Zauber Adaption

::

### Arkane Sinnesschärfe

> **1 Adaption / Zauber**

:

*Wenn ein Fernangriff nicht auf dich Zielt:* Greife das Geschoss mit einem Zaubertrick von dir an. Prüfung wird bestimmt durch die Größe des Geschosses.

:

### Magiefluss

> **2 Adaption / Intelligenz | Charisma < 11**

:

*Wenn du Ziel eines Zaubers bist und ein Ziel im Nahkampf hast:* Das Ziel muss von dir Berührt werden. Das Ziel bekommt die Hälfte des Schadens.

:

### Arkaner Instinkt

> **1 Adaption / Zauber**

:

*Wenn ein Angriffswurf von einem Zauber misslingt:* Du kannst **1d4** zum Angriffswurf hinzufügen.

:

### Schimmernde Konzentration

> **1 Adaption / Konzentration Zauber**

:

*Wenn du ein Konzentrationszauber halten willst:* Du bist im Vorteil.

:

## Geschicklichkeits Adaption

::

### Rückzug

> **X Adaption / Nahkampf / Geschicklichkeit > 11**

:

*Nach einem misslungenden Nahkampfangriff:* Bewege dich **X * 3** Felder ohne ein Gelegenheitsangriff vom Ziel.

:

### Sprungbeine

> **1 Adaption / Geschicklichkeit > 13**

:

*Benutze ein Hindernis in 2 Felder Entfernung:* Springe zu einem Feld das 2 Felder entfernt ist von diesem Hindernis. (Dies kann auch für *Tot von Oben* benutzt werden.)

:

## Wissen Adaption

::

### Schwachpunkt Erkennen

> **1 Adaption / Weisheit > 13**

:

Du kannst einen Schwachpunkt eines Gegners erkennen. Bei **Misserfolg** bekommst du `+1` auf deinen nächsten Angriff, durch das analysieren des Gegners. 

:

### Reaktive Taktik

> **1 Adaption**

:

*Wenn du ein Rettungswurf nicht schaffst:* Zähle **+2** dazu.

:

## Combo Adaption

### Combo: Tödliche Gelegenheit

> **2 Adaption**

:

*Wenn ein Verbündeter einen kritischen Treffer landet:* Du hast für 2 Züge Vorteil gegen dieses Ziel.

:

### Combo: Offene Flanke

> **2 Adaption / Nahkampf**

:

*Wenn ein Verbündeter in den Nahkampfbereich kommt mit dem du auch im Nahkampf stehst:* Du darfst einen Angriff auf das Ziel machen.

:

> **2 Adaption / Nahkampf**

:

*Wenn du in den Nahkampf springst, von einer Position oberhalb des Ziels:* Du hast Vorteil für einen Angriff. Zusätzlich macht du **1d6** mehr Schaden pro Level.

# Sanguine Adept - Zauberer

::

## Passiv Fähigkeiten

::

### Sanguine

Du kannst so viel `Sanguine` besitzen wie dein Level.
Bei einer kurzen und langen Rast, fülle dein `Sanguine` auf bis zur hälfte der Kapazität (Abgerundet).

### Sanguine Resonanz

Immer wenn ein Gegner im Kampf mehr Schaden nimmt als `5 + [Level]` erstelle 1 Sanguina.

## Fähigkeiten

::

### Blutpfeil



> **Aktion / Sanguina: 1 / Reichweite: 6 Felder / Angriff**

:

Das Ziel nimmt `1d8 + [CHA]` Schaden. Am Ende des Zugs des Ziels: Das Ziel nimmt `1d4` Schaden.

> *Erstelle einen Pfeil aus Blut der ein Ziel niederstreckt.*

:

### Blut Command

> **Aktion / Sanguina: X / Reichweite: Y / Zauber**

:

Wähle ein Effekt für ein Ziel in Hörreichweite:

- Heile das Ziel um `X * 2 + [CHA]`
- Entwaffne das Ziel. Das Ziel hat die Möglichkeit eine **Weisheits Rettungswurf** zu machen. Der Rettungswurf wird um `X` schwerer.

:

### Blutdornen

> **Bonus / Reichweite: 1 Feld / Nahkampf / Finess**

:

Greife einen Gegener an und füge `1d4 + [Sanguina]` Schaden zu. Erhalte außerdem `1 Adaptionspunkt`, wenn du triffst.

:

\column

### Blutranken

> **Aktion / Sanguina: 1 / Reichweite: 6 / Zauber / Konzentration**

:

Bestimmen von einem Ziel eine Kette von 2 mal 5 Feldern. Auf diesen wachsen blutige Ranken. 
Jeder der die Ranken berührt nimmt `1d8 + [Sanguina]` Schaden und kann einnen **Geschicklichkeits Rettungswurf** bestehen um nur die Hälfte des Schadens zu bekommen.

> *Nutze die Ehemalige Lebenskraft einer Leiche um die verbliebenden zu Dezimieren. Dunkle Blutranken entfesseln sich aus der Leiche.*


### Blutparade

> **Reaktion / Sanguina: 1 / Reichweite: 1 / Nahkampf**

:

Verringere den Schaden um `1d6`. Würfelst du den Maximalen Schaden: Erhalte 3 `Sanguina` und der Angreifer erleidet Schaden in höhe `[Sanguina]`.

:

### Bluthagel

> **Aktion & Bonus / Sanguina: X (MIN: 2) / Reichweite 5 / Zauber**

:

Treffe alle Ziele in einem Umkreis von `X`. Jedes Ziel kann einen **Geschicklichkeits Rettungswurf** bestehen um den Schaden zu halbieren. Der Schaden ist: `[X]d6 + [Sanguina]`.

> *Manifestiere 1000 Pfeile aus Blut die über den Gegner regnen.* 

:

### Essenz: Spiegelzauber

> **Aktion & Bonus / Reichweite: 6 / Zauber / Konzentration (Max: 6)**

:

Erstelle einen Spiegelraum. Wenn du 1 Zauber oder Zaubertrick benutzt: Führe deinen Angriff **1d4** mal aus.

:

# Rift Striker - Rouge

::

## Passiv Fähigkeiten

::

### Riftflux

:

Maximal kannst du `[Level] + [Level] / 3 Riftflux` besitzen.
Erhalte bei einer kurzen Rast 2 `Riftflux` und bei einer langen Rast 4 `Riftflux`.

:

### Rift Beherrscher

:

Jeden Zug kannst du `1d6` würfeln. Ist das Ergebnis eine 6, erhälst du einen `Riftflux`.
Befindest du dich in Dunkelheit, dann würfle `2d6`.

:

## Fähigkeiten

::

### Rift Strike

> **Aktion / Riftflux: 1 / Reichweite 5 / Nahkampf**

:

Benutze 1 von diesen Fähigkeiten:
- Du Springst an einen Gegner in Reichweite 5 und greifst ihn an.
- Nach einem Angriff entferne dich um 5 Felder im Frontark des Gegners.

:

### Rift Einstimmung

> **Bonus / Zauber**

:

Würfle `1d4` bei einer 4 erhälst du 1 Riftflux und der nächste Angriff auf dich, bis zu deinem nächsten Zug, ist im Nachteil.

:

### Riftchorda 

> **Aktion / Riftflux: X / Reichweite: 2 / Nahkampf**

:

Greife `X` mal einen Gegner in 2 Reichweite an. Füge mit jedem Angriff, der auf ein neues Ziel geht, `1d6` Schaden hinzu.

:

\column

### Rift Parade

> **Bonus & Reaktion / Riftflux: 1 / Zauber**

:

Schaffst du einen **Geschicklichkeits Rettungswurf** gegen den Angriffswurf des Gegners, dann kannst du den Angriff auf ein anderes Legitimes Ziel werfen aus deiner Sichtweise, es darf nicht der Angreifer sein.

> *Erschaffe ein Rift vor dem Angriff und ein gespiegelten Rift an gleicher stelle.*

:

### Rift Hinterhalt

> **Aktion / Riftflux: 2 / Reichweite 5 / Nahkampf**

:

Du springst in den Rücken eines Gegners. Hat das Ziel dich in der letzten Runde nicht angegriffen, dann hast du Vorteil.

:

### Rift Spalte

> **Aktion / Riftflux: X / Zauber**

:

Schicke X Gegner die sich in einem Kegel von 3 Felder in deinem Frontark befinden in die Luft sie nehmen `1d10` Fallschaden.

> *Ramme dein Schwert in den Boden und erschaffe einen Rift der deine Gegner in den Himmel Teleportiert.*

:

### Rift Meteor

> **Aktion & Bonus / Riftflux: X (Min: 3) / Zauber**

:

***Um diese Fähigkeit nutzen zu können muss die Umgebung in Verbindung stehen mit der Rift Welt.***
Bewirke in einem Sichtbaren Ziel Schaden in höhe von `8d6 Feuer + 8d6 Rift` Schaden. Die Reichweite der Explosion bestimmt der `Schaden / 10`. Alle Ziele die die Explosion direkt sehen können, erhalten `3d6` Schaden.

:

# Zenruptor - Monk

::

## Passive Fähigkeiten

::

### Haltung

:

Du hast die Auswahl von 3 Haltungen.
- **Neutral:** Diese Haltung besitzt du immer am Anfang eines Kampfes.
- **Ruhe:** Solange du in Ruhe verringerst du jeden Schaden den du nimmst oder austeilst um `1d6`.
- **Zorn:** Solange du in Zorn bist füge zu jedem Schaden den du bekommst oder austeilst `1d6` hinzu.
- **Aufgestiegen:** Du erhällst 2 Aktionen und 2 Bonusaktionen, am Ende deines Zuges gehst du in die Haltung **Neutral**. Außerdem kannst du alle Haltungs Bedingungen Ignorieren.

:

### Mantra

:

Eine Resource die am Anfang des Kampfes um 1 reduziert wird.

:

## Fähigkeiten

::

### KI Einstimmung

> **Bonus / Ruhe | Neutral / Zauber**

:

Wenn du nicht in `Zorn` bist, bekomme `1 KI` Punkt. Setze dich in die `Ruhe` Haltung.

:

### Elemental Fury

> **Aktion / Zauber**

:

Ein Gegner in deiner Reichweite bekommt einen gelegenheits Angriff. Gehe in die `Zorn` Haltung.

:

### Elemental Fracture

> **Bonus / Zorn / KI: 1 / Nahkampf**

:

Wenn das Ziel einen **Konstituion Rettungswurf** ablegen, schafft das Ziel es nicht, verliert das Ziel bis zu deiner nächsten Runde `5 AC`.

:

\column

### Phoenix Faust

> **Aktion / Zorn / KI: 1 / Zauber**

:

Greife alle Gegner in einer Geraden an (Felder: 4). Füge jedes Ziel `1d8 + [WEI]` Schaden zu, die Ziele haben die Möglichkeit einen **Geschicklichkeits Rettungswurf** um den Schaden zu halbieren. 

### Offene Woge

> **Aktion / Ruhe / KI: 1 / Nahkampf**

:

Treffe bis zu 3 Gegner im Nahkampf. Ignoriere den Nachteil deiner `Ruhe` Haltung.

:

### Leere Faust

> **Bonus / Ruhe | Zorn / Nahkampf**

:

Greife einen Gegner an der dir letzte Runde Schaden gemacht hat. Schaden: `1d6 + [WEI]`. Gehe in die Haltung `Neutral` und erhalte `1 Mantra`.

:

### Leerer Tritt

> **Bonus / Ruhe | Zorn / Nahkampf**

:

Greife einen Gegner an, dem du in diesen Zug Schaden zugefügt hast. Schaden: `1d6 + [WEI]`. Setzt du `1 KI` Punkt ein: muss das Ziel ein **Geschicklichkeits Rettungswurf** machen. Schafft das Ziel es nicht dann füge zusätzlich `1d6` Schaden zu und das Ziel fällt hin. Gehe in die Haltung `Neutral` und erhalte `1 Mantra`.

:

### Leerer Verstand

> **Reaktion / Ruhe | Zorn / Zauber**

:

Wirst du von einem Zauber getroffen kannst du deinen **Rettungswurf** um `1d4` erhöhen. Gehe in die Haltung `Neutral` und erhalte `1 Mantra`.

:

### Transzendent

> **Aktion & Bonus | Mantra: 4 | Zauber**

:

Gehe in die Haltung `Aufgestiegen`. Dein Schaden wird um jeden weiteren Mantra den du einsätzt um `1d8` erhöht in diesem Zug.

:

# Architekt des Bane - Druid

::

## Passive Fähigkeiten

::

### Schädel Trophäe

:

Du erhälst einen **physischen Angriff** von der Kreatur des Kopfes als **Bonus Aktion**.

:

### Trophäen Gürtel

:

Du kannst eine Trophäe aus einem Kampf mitnehmen in den nächsten Kampf. Nach einer **Langen Rast** ist die Trophäe verbraucht.

:

## Fähigkeiten

::

### Schnelle Ernte

> **Aktion / Felder: 5 / Zauber**

:

Du kannst aus einer Leiche eine Zufällige Trophäe ernten. Die Leiche wird zerstört. Würfle `1d4`, bei `1` der **Arm**, bei `2` das **Bein**, bei `3` der **Körper**, bei `4` der **Schwanz *(Nur bei Monstern mit Schwanz)***.

:

### Ernte des Bane

> **Aktion / Felder: 1 / Nahkampf**

:

Du kannst aus einer Leiche eine Trophäe erhalten. **Schädel**, **Arm**, **Bein**, **Körper** oder **Schwanz *(Nur bei Monstern mit Schwanz)***.

:

### Banes Bastard

> **Aktion & Bonus / Trophäe: X / Felder: 1 / Zauber**

:

Erschaffe einen Leichendiener aus deinen Trophäen. Leben: `1d6 + 1` mal `[Trophäen]`. Schadenbonus: `[Trophäen]`. Die `RK` ist `12 + [Trophäen]`. Jeder Trophäentyp bringt einen eigenen Bonus.

:

\column

### Marionette des Bane

> **Aktion / Trophäe: 1 / Dauer: 3 / Zauber**

:

Wähle ein Ziel in Reichweite. Das Ziel kann jede Runde am Ende ein **Charisma Rettungswurf** machen um den Zauber zu entkräften. Du kannst eine **Aktion** des Ziels ausführen mit dem Körperteil der Trophäe.

:

### Echo des Endes

> **Aktion / Trophäe: 1 / Dauer 10 / Zauber / Konzentration**

:

Zerstöre eine `Trophäe` und erhalte einen den passenden Zauber. Der Zauber kann wiederholt werden als `Bonus` Aktion:
- **Schädel:** Das Ziel darf die `Bonus` Aktion als Normale Aktion nutzen. Nach Benutzung muss das Ziel einen **Weisheits Rettungswurf** schaffen, wenn nicht dann greift das Ziel mit dem halben Schaden, welches er in diesem Zug verursacht hat, sich selbst an. (Min: 6).
- **Arm:** Das Ziel kann doppelt so weit Werfen; +1 Angriffswurfe; + die hälfte des Schadens.
- **Bein:** Das Ziel kann doppelt so weit laufen und springen. `RK +1` 
- **Körper:** Das Ziel erhält `RK +3`. 
- **Schwanz:** Das Ziel kann eine `Aktion` ausführen vom Monster des Schwanzes. Das Ziel muss nach der Aktion einen **Weisheits Rettungswurf** schaffen, oder es wird **Wahnsinnig**.

:

### Rot hallendes Sanktuar

> **Aktion & Bonus / Trophäe: >4 / Dauer: X / Zauber / Konzentration**

:

Erschaffe einen Bannkreis im Radius 2. Im Bannkreis wirkt folgendes:
- **Verbündet:**
  - Zauber benötigen keine Simple Resource.
  - Angriffs- und Zauberwürfe: `+2`; `RK +2`
  - Erhält eine `Bonus` Aktion
- **Feind:**
  - Im Umkreis von 2: **Weisheits Rettungswurf**, sonst **Verängstigt**
  - Bewegung Halbiert

:
